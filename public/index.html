Error Reference
400
Bad Request — name field is missing or empty in the request body
422
Unprocessable Entity — name field is the wrong type (e.g. a number)
404
Not Found — no profile exists with the given UUID
502
Bad Gateway — Genderize, Agify, or Nationalize returned invalid or null data
500
Internal Server Error — unexpected server-side failure
Classification Rules
Age Group (from Agify)
0 – 12 → child
13 – 19 → teenager
20 – 59 → adult
60+ → senior
Nationality (from Nationalize)
The country with the highest probability value in the response array is chosen as country_id.
